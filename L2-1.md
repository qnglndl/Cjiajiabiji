# 2-1 for循环语句&for循环与变量
## 1.for循环语句
例:
```cpp
for(int i = 1;i <= 100;i++>){
    cout << "!"
}
```

```cpp
int i = 1;   //循环的起点
```
```cpp
i <= 100; //循环终点
```
```cpp
i++; // 循环变量多长
```
```cpp
{
    ... //循环内容
}
```
<span style="color:red">注意：</span>
1. <span style="color:red"> for后面括号内有两个";"间隔 </span>
2. <span style="color:red"> 循环终点有"="则包含终点 </span>

# 3.执行顺序
flowchart TD
    A([开始]) --> B[①]
    B --> C{判断条件②}
    C -- 不成立 --> D([结束])
    C -- 成立 --> E[④]
    E --> F[③]
    F --> B
