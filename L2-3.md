# 1.for和if的并列
```cpp
for(    ;   ;   ){

}
if(    ){

}
else{

}
```
for与if的并列，可用于先求和在对求和结果做判断
---

### 应用举例：判断从a到b的所有数字的和是否为6的倍数
```cpp
int a,b,sum = 0;
cin >> a >> b;
for(int i = a;i <= b;i++){
    sum += i;
}
if(sum %6 == 0){
    cout << "yes";
}
else{
    cout << "no";
}
```
